# 术语通 TermHub - 专业英语词汇学习系统


## 简介

**术语通（TermHub）** 是一款专业英语词汇学习系统，专注于网络安全、软件开发、数据库等技术领域的专业词汇。

- 🎯 **专业聚焦**：覆盖网络安全、Web渗透、密码学、数据库、操作系统等多个技术领域
- 🤖 **AI 赋能**：AI 生成例句、AI 补全单词、AI 智能管理词库
- 🔊 **真人发音**：微软语音引擎实时生成英式/美式发音，自然流畅
- 📱 **响应式设计**：完美适配电脑、平板、手机
- 💾 **本地部署**：数据存储在本地 SQLite 数据库，安全隐私
- ⚙️ **管理后台**：内置完整的词库管理功能，支持批量导入导出
- 📋 **操作日志**：完整记录所有管理操作，支持归档管理

> 💡 本项目由 [豆包 AI](https://www.doubao.com/) 辅助开发，由 Half-Skilled TvT 负责产品设计、需求定义和质量把控。

## 快速启动

### Windows 用户
双击 `启动网站.bat` 即可自动启动

### Mac/Linux 用户
打开终端，进入项目文件夹，执行：
```bash
chmod +x 启动网站.sh
./启动网站.sh
```

## 手动启动

如果脚本无法运行，也可以手动启动：

1. 安装依赖：
```bash
pip install flask edge-tts
```

2. 运行网站：
```bash
python app.py
```

3. 浏览器打开：http://localhost:5001/

## 语音功能说明

- 点击音标旁的 🔊 按钮即可播放发音
- 英式发音：en-GB-SoniaNeural（女声，标准英音，清晰自然）
- 美式发音：en-US-JennyNeural（女声，柔和美音，友好体贴）
- **实时生成**：点击时实时生成语音，不占用存储空间
- **浏览器缓存**：生成过的语音会自动缓存30天，第二次点击秒播
- **加载提示**：生成过程中显示「...」滚动动画，提示正在加载

- 如果语音生成失败，会自动回退到浏览器自带的语音合成功能

## AI例句功能
- 登录后，每个单词卡片都会显示「🤖 AI例句」按钮

- 需要先在管理后台「设置 → AI设置」中配置API Key

- **浏览器缓存**：生成过的例句自动缓存30天，第二次打开秒开，节省API调用

- **安全优化**：未登录时不输出任何管理功能的HTML元素，真正的权限控制在后端API层面，双重保障

## 数据库结构

数据库使用 SQLite，包含以下八张表：

### categories（分类表）
- id：分类ID（主键，自增）
- name：分类名称（唯一）
- created_at：创建时间

### vocabulary（单词表）
- id：单词ID（主键，自增）
- category_id：分类ID（外键，关联 categories.id）
- english_word：英文单词
- british_phonetic：英式音标
- american_phonetic：美式音标
- chinese_definition：中文释义
- created_at：创建时间

### users（用户表）
- id：用户ID（主键，自增）
- username：用户名（唯一）
- password：密码（MD5加密存储）
- created_at：创建时间

### logs（日志表）
- id：日志ID（主键，自增）
- action：操作类型
- description：操作描述
- ip_address：IP地址
- created_at：创建时间
- is_archived：是否归档（0:未归档，1:已归档）

### site_settings（站点设置表）
- id：设置ID（主键，自增）
- site_name：站点名称
- site_description：站点描述
- site_keywords：站点关键词
- site_copyright：版权信息
- site_icp：备案号
- updated_at：更新时间

### ai_settings（AI配置表）
- id：配置ID（主键，自增）
- api_key：API密钥
- api_url：API地址
- model_name：模型名称
- system_prompt：系统提示词
- updated_at：更新时间

### ai_chat_history（AI聊天历史表）
- id：记录ID（主键，自增）
- role：角色（user/assistant）
- content：消息内容
- created_at：创建时间

### ai_operations（AI操作记录表）
- id：记录ID（主键，自增）
- operation_type：操作类型
- description：操作描述
- details：详细信息（JSON格式）
- status：状态（success/failed）
- created_at：创建时间

## 常见问题

**Q：提示找不到Python？**
A：请先安装Python 3.8或更高版本，下载地址：https://www.python.org/downloads/

**Q：依赖安装失败？**
A：可能是网络问题，可以试试换国内源：
```
pip install flask edge-tts -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**Q：浏览器打不开？**
A：确认网站已经启动，然后访问 http://localhost:5001/

**Q：语音播放没有声音？**
A：请检查以下几点：
1. 确认电脑音量已打开
2. 确认已联网（语音需要实时生成）
3. 可能是浏览器自动播放限制，点击页面任意位置后再试
4. 如果还是不行，系统会自动回退到浏览器自带的语音合成功能

**Q：语音播放第一次慢，第二次快？**
A：这是正常的。第一次点击时需要实时生成语音，生成后会自动缓存到浏览器，第二次及以后点击同一个单词会直接从缓存读取，秒播。

**Q：管理后台怎么进？**
A：首页左侧边栏底部有「⚙️ 管理后台」链接，点击即可进入。或者直接访问 http://localhost:5001/admin

**Q：管理后台的账号密码是什么？**
A：默认账号密码都是 admin。登录后可以在「设置」页面修改密码。

**Q：删除分类会怎么样？**
A：删除分类会同时删除该分类下的所有单词，此操作不可撤销，请谨慎操作。

**Q：怎么添加新分类？**
A：有两种方式：
1. 在管理后台的「分类管理」中点击「添加分类」
2. 在添加单词时，直接输入新的分类名，保存后会自动创建该分类

**Q：怎么合并分类？**
A：在分类管理页面，勾选要合并的分类，然后点击「合并分类」按钮，选择目标分类或输入新分类名即可。

**Q：AI助手怎么用？**
A：在管理后台左侧点击「🤖 AI助手」进入聊天页面，用自然语言告诉AI你要做什么即可。比如"添加一个单词test，意思是测试，分类到其他"。

**Q：AI助手需要配置什么？**
A：需要在「设置」→「AI设置」中配置你的API Key和API地址。支持所有兼容OpenAI格式的API接口。

**Q：AI操作会记录吗？**
A：会的，所有AI执行的操作都会记录在操作日志中，可以在「日志管理」页面查看。

**Q：AI能做什么操作？**
A：AI可以帮你查询、添加、修改、删除单词和分类，支持批量操作和合并分类。你只需要用自然语言描述，AI会自动理解并执行。

**Q：日志归档有什么用？**
A：归档可以把旧的日志移到归档列表，让正常日志列表更清爽。归档的日志可以查看、取消归档或永久删除。超过90天的日志会自动归档。

**Q：首页可以直接编辑单词吗？**
A：可以。登录后，首页每个单词卡片上都会显示「✏️ 编辑」和「🗑️ 删除」按钮，可以直接操作，无需进入管理后台。

**Q：搜索结果能看到单词属于哪个分类吗？**
A：可以。搜索时每个单词卡片上都会显示它的分类标签，方便你知道它属于哪个分类。

**Q：未登录时能看到管理功能吗？**
A：不能。未登录时，页面不会输出任何管理功能的HTML代码，真正的权限控制在后端API层面，双重保障安全。

**Q：删除单词后，左侧分类的数量会更新吗？**
A：会的。删除或编辑单词后，左侧分类菜单的单词数量会自动刷新，保持实时准确。
